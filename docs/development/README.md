# Development Documentation

This folder contains all development-related documentation for the StoreSync project.

## Structure

- `current-status.md` - Current project status and progress
- `architecture.md` - System architecture and design decisions
- `shopee-integration/` - Shopee integration specific documentation
- `database/` - Database schema and related documentation
- `troubleshooting/` - Error handling and troubleshooting guides
- `next-steps.md` - Planned development steps and roadmap
